# AI Expense Tracker

A full-stack web application to track and analyze personal expenses.

### Features

- User authentication (Register/Login)
- Add, edit, delete expenses
- Category-wise analytics
- Daily spending visualization
- Monthly budget tracking
- Financial score calculation
- Spending prediction
- CSV export functionality
- Dark mode support

### Tech Stack

Frontend:
- HTML
- Tailwind CSS
- Chart.js

Backend:
- Python (Flask)
- SQLAlchemy ORM

Database:
- SQLite

### How to Run

1. Install dependencies:
   pip install flask flask_sqlalchemy werkzeug

2. Run the application:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
